# Python Mini Games

A collection of small Python games I'm building and improving while learning. Each one starts simple and gets refined over time as I pick up new concepts.

## Games

### Guess the Number
A classic number-guessing game. The program picks a random number between 1 and 100, and you have 10 attempts to guess it — with hints telling you if you're too high or too low.

**How to run:**
```bash
python guess_the_number.py
```

**Concepts used:** `random` module, `while` loops, conditionals, f-strings
